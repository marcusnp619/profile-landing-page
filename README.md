# 🎯 Security Researcher Portfolio - Neon Techy Edition

An epic, futuristic cybersecurity profile landing page built with neon aesthetics, featuring an interactive mock terminal, bounty visualizer, and comprehensive security toolkit showcase.

## 🌟 Features

### 💻 Auto-Play Mock Terminal
- **Star Feature**: Realistic hacking session simulation with typewriter timing
- **Interactive Commands**:
  - `help` - Lists all available commands
  - `scope` - Scope definition from security best practices
  - `report` - Report writing tips and best practices
  - `tools` - Full toolkit list and descriptions
  - `platforms` - All bounty platforms with links
  - `clear` - Reset terminal output
  - `hack` - Restart auto-play simulation
- **Command History**: Navigate with arrow keys (↑↓)
- **Auto-starts** on page load

### 🛠️ Core Toolkit Spotlight
- **Kali Linux** - Penetration testing OS with 600+ tools
- **Burp Suite** - Industry-leading web app security testing
- **Nmap** - Network reconnaissance and security auditing
- Color-coded cards with feature tags and clickable links to exact sites

### 💰 Bounty Payout Visualizer
- Animated CSS bar chart with smooth scroll triggers
- Severity levels: Critical ($7,200) → Low ($250)
- Gradient color animation with glow effects

### 🌐 Bounty Platforms Grid
12 major platforms with color-coded badges and direct links:

- Immunefi
- YesWeHack
- Federacy
- Synack

### ⚖️ Legal Landscape

- Responsible disclosure guidance

## 🛠️ Quick Start

1. Clone the repository:
```bash
git clone https://github.com/marcusnp619/profile-landing-page.git
cd profile-landing-page
```

2. Open in browser:
```bash
# Simply open index.html in your browser
open index.html
# or
firefox index.html
```

3. Or deploy to GitHub Pages:
- Go to repository Settings → Pages
- Select `main` branch as source
- Site will be live at: `https://marcusnp619.github.io/profile-landing-page/`

## 💡 Terminal Commands Reference

### `help`
Displays all available commands with descriptions.

### `scope`
Shows the scope definition framework for authorized security testing.

### `report`
Provides best practices for writing security reports.

### `tools`
Lists the full toolkit with descriptions.

### `platforms`
Shows all bounty platforms with direct links.

### `clear`
Clears the terminal output.

### `hack`
Restarts the auto-play hacking simulation.

## 🎯 Terminal Auto-Play Sequence

The terminal auto-plays on page load:
1. `whoami` command
2. `nmap -sV` network scan simulation
3. `burpsuite --scan` vulnerability detection
4. Shows realistic security findings

After 10 seconds, you can interact with the terminal.
